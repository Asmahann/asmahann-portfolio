# Portfolio site for Asmahan Umer

This repository contains a simple static portfolio site that dynamically loads projects from `projects.json`.

Preview locally:
1. Clone the repo.
2. In the repo folder, run a static server. Examples:
   - Python 3: `python -m http.server 8000`
   - Or use Live Server in VS Code.
3. Open `http://localhost:8000` in your browser.

Customize:
- Edit `projects.json` to add correct `url` values (repo links) and `demo` links or images.
- Modify text in `index.html` to update hero/about/contact.
- Replace styling in `styles.css` or add images in an `assets/` folder.

Deploy:
- Push to a GitHub repo and enable GitHub Pages (Settings → Pages → Deploy from main branch / root).
- Or deploy to Netlify / Vercel by pointing at this repo.
